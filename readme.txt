The Case LMS plugin is a Legal Management System, a simple way to organize and manage legal documents.

== Description ==

CaseLMS is a Legal Management System designed for WordPress. Our plugin transforms your WordPress site into a powerful legal document management and organizational tool.
It's perfect for law firms, solo practitioners, legal departments, law schools, and anyone who needs to manage a wide array of legal documents efficiently.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. To get started you will need to create the slides for the slider in WordPress admin menu select Add New under Slim Slides.
